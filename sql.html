Covid 19 Data Analysis

Using a range of SQL queries, I conducted a comprehensive analysis of COVID-19 data from 2021. I calculated the total number of cases and deaths for each country and identified the highest infection rates relative to the population. I also extracted global case insights to understand broader trends and compared the total number of vaccinations completed across various regions. This analysis provided a detailed overview of the pandemic's impact and vaccination progress on a global scale.

Coffee Shop Sales Analysis

I utilized my SQL skills to extract key performance indicators such as total sales, total number of orders, and quantity sold, and compared these metrics to the previous month. Additionally, I analyzed these indicators on a daily basis, comparing weekend versus weekday sales, as well as sales by location and product category. I also identified the best-selling product of the month. Leveraging these insights, I built a Power BI dashboard that provides an overview, aiding in informed decision-making for future strategies.
